# PostureFit PRD — 火山杯Agent创新大赛黑客松需求文档

> **项目名称**：PostureFit — AI体态矫正运动搭子  
> **赛道**：火山杯Agent创新大赛 · 生活赛道 · 运动子赛道  
> **版本**：v3.0（黑客松24小时MVP版）  
> **目标**：在24小时内开发一个可演示的网页应用，聚焦体态矫正，通过AI识别体态问题并提供个性化训练计划。  
> **文档面向**：3位开发同学（前端A、前端B、全栈C）

---

## 1. 项目背景与痛点

### 1.1 市场背景
中国有超过4亿久坐人群，包括程序员、学生、办公室白领。每天久坐8-12小时导致颈椎前倾、圆肩驼背、腰椎问题年轻化（20-35岁发病率逐年上升）。体态问题不仅影响外观，更导致慢性疼痛、运动损伤风险增加。传统解决方案（医院康复科、健身私教、网络教程）存在高成本、碎片化、缺乏个性化等痛点。

### 1.2 用户痛点
- **价格门槛高**：私教课年费万元级，医院康复单次200-500元。
- **时间成本大**：需要预约、通勤，难以融入忙碌生活。
- **缺乏个性化**：通用教程无法针对个人体态问题和身体状态（如产后、生理期、疲劳状态）。
- **无监督机制**：缺乏持续激励和反馈，容易放弃。
- **信息碎片化**：网络教程质量参差不齐，无法判断安全性。

### 1.3 为什么需要AI搭子
AI技术可以解决上述痛点：通过摄像头实时识别体态问题，生成个性化训练计划，根据用户反馈动态调整，并提供教练式互动激励。PostureFit旨在成为每个人的“AI体态矫正运动搭子”，让专业体态矫正变得可及、个性化、可持续。

**需求**：希望有AI教练指导，动作简单易学，能看到体态改善趋势，获得成就感。

## 6. 关键算法说明

### 6.1 PoseNet关键点索引
PoseNet返回17个关键点，索引如下：
- 0: 鼻子
- 1: 左眼
- 2: 右眼
- 3: 左耳
- 4: 右耳
- 5: 左肩
- 6: 右肩
- 7: 左肘
- 8: 右肘
- 9: 左腕
- 10: 右腕
- 11: 左髋
- 12: 右髋
- 13: 左膝
- 14: 右膝
- 15: 左踝
- 16: 右踝

### 6.2 三点夹角计算函数
计算三个点形成的夹角（以度为单位）：
```javascript
function calculateAngle(pointA, pointB, pointC) {
  // pointA, pointB, pointC 为 {x, y} 坐标
  const vectorBA = { x: pointA.x - pointB.x, y: pointA.y - pointB.y };
  const vectorBC = { x: pointC.x - pointB.x, y: pointC.y - pointB.y };
  
  const dotProduct = vectorBA.x * vectorBC.x + vectorBA.y * vectorBC.y;
  const magnitudeBA = Math.sqrt(vectorBA.x ** 2 + vectorBA.y ** 2);
  const magnitudeBC = Math.sqrt(vectorBC.x ** 2 + vectorBC.y ** 2);
  
  const cosAngle = dotProduct / (magnitudeBA * magnitudeBC);
  const angleRad = Math.acos(Math.max(-1, Math.min(1, cosAngle))); // 防止浮点误差
  const angleDeg = angleRad * (180 / Math.PI);
  
  return angleDeg;
}
```

### 6.3 头前伸角简化计算
头前伸角（Forward Head Angle）定义为耳垂、肩峰、颈椎的夹角。简化计算：
```javascript
function calculateForwardHeadAngle(ear, shoulder, neck) {
  // 使用左耳(3)、左肩(5)、鼻(0)作为近似
  // 实际中neck可取鼻和肩的中点
  return calculateAngle(ear, shoulder, neck);
}
```
**正常范围**：< 5°（轻度前伸）  
**异常阈值**：> 10°（中度前伸）

### 6.4 圆肩角简化计算
圆肩角（Rounded Shoulder Angle）定义为肩峰、肘部、髋部的夹角。简化计算：
```javascript
function calculateRoundedShoulderAngle(shoulder, elbow, hip) {
  // 使用左肩(5)、左肘(7)、左髋(11)
  return calculateAngle(shoulder, elbow, hip);
}
```
**正常范围**：< 20°  
**异常阈值**：> 30°（明显圆肩）

### 6.5 骨盆前倾角简化计算
骨盆前倾角（Anterior Pelvic Tilt Angle）定义为髋部、膝部、踝部的夹角。简化计算：
```javascript
function calculateAnteriorPelvicTiltAngle(hip, knee, ankle) {
  // 使用左髋(11)、左膝(13)、左踝(15)
  return calculateAngle(hip, knee, ankle);
}
```
**正常范围**：5-15°  
**异常阈值**：> 20°（明显前倾）

**注意**：这些是简化计算，实际体态评估需要更复杂的生物力学模型。在24小时黑客松中，使用这些近似值足以演示概念。

## 7. Coze Bot设计详细规格

### 7.1 输入变量列表
| 变量名 | 类型 | 示例 | 说明 |
|--------|------|------|------|
| `forwardHeadAngle` | number | 12.5 | 头前伸角度 |
| `roundedShoulderAngle` | number | 35.2 | 圆肩角度 |
| `anteriorTiltAngle` | number | 22.8 | 骨盆前倾角度 |
| `coachStyle` | string | "encouraging" | 教练风格：encouraging/strict/humorous |
| `coachGender` | string | "female" | 教练性别：male/female |
| `userGoal` | string | "改善圆肩" | 用户目标描述 |
| `bodyState` | string | "normal" | 身体状态：normal/postpartum/menstrual/fatigued/teenager |
| `feedback` | string | "做完了" | 用户反馈（打卡时） |

### 7.2 工作流步骤
1. **接收输入**：解析角度数据、教练风格、用户目标、身体状态。
2. **问题判断**：根据角度阈值判断主要体态问题（头前伸/圆肩/骨盆前倾）。
3. **动作匹配**：从预置动作库中选择针对该问题的3个动作。
4. **计划生成**：生成每日训练计划，包含动作名称、描述、时长、B站搜索链接。
5. **风格适配**：根据教练风格调整输出语气。
6. **反馈处理**：接收用户反馈，调整后续计划（如“太累了”则降低强度）。

### 7.3 动作库（预置列表）
| 体态问题 | 动作1 | 动作2 | 动作3 |
|---------|-------|-------|-------|
| **头前伸** | 颈部后缩训练 | 上斜方肌拉伸 | 颈深屈肌强化 |
| **圆肩** | 胸肌拉伸 | 菱形肌强化 | 肩胛骨后缩 |
| **骨盆前倾** | 骨盆后倾训练 | 臀桥 | 髋屈肌拉伸 |

**每个动作包含**：
- 名称
- 描述（简短说明）
- 时长（秒）
- B站搜索链接（如：`https://search.bilibili.com/all?keyword=颈部后缩训练`）

### 7.4 输出格式
Coze Bot返回纯文本 + Markdown链接，示例：
```
根据您的体态分析，您存在明显的圆肩问题（角度35°）。我为您设计了今日训练计划：

1. **胸肌拉伸**（60秒）  
   描述：双手背后交叉，挺胸拉伸胸肌。  
   视频参考：[B站搜索](https://search.bilibili.com/all?keyword=胸肌拉伸)

2. **菱形肌强化**（45秒）  
   描述：俯卧，双手后伸，抬起上半身强化背部肌肉。  
   视频参考：[B站搜索](https://search.bilibili.com/all?keyword=菱形肌强化)

3. **肩胛骨后缩**（60秒）  
   描述：坐直，双肩向后夹紧肩胛骨。  
   视频参考：[B站搜索](https://search.bilibili.com/all?keyword=肩胛骨后缩)

完成后请回复“做完了”，如果感到疲劳请回复“太累了”。
```

### 7.5 风格切换System Prompt示例
**鼓励风格**：
```
你是一位温柔鼓励的AI体态教练。用户可能存在体态问题，但你要给予积极的支持和鼓励。使用亲切的语气，多使用“加油”、“你很棒”、“坚持就是胜利”等鼓励性话语。在用户反馈疲劳时，给予理解和调整建议。
```

**严厉风格**：
```
你是一位严格专业的AI体态教练。你以结果为导向，要求用户严格执行训练计划。使用简洁、直接的指令，避免过多情感表达。在用户反馈疲劳时，强调坚持的重要性，但也要提供科学的恢复建议。
```

**幽默风格**：
```
你是一位幽默风趣的AI体态教练。你用轻松幽默的方式指导训练，使用比喻、玩笑让用户感到有趣。在用户反馈疲劳时，用幽默的方式鼓励他们继续。
```

## 8. 交互流程

### 8.1 完整步骤
1. **打开网页**：进入PostureFit首页，展示产品介绍和“开始评估”按钮。
2. **选择拍摄模式**：用户选择全身/半身/坐姿模式。
3. **拍照**：摄像头实时预览，用户点击“拍照”按钮。
4. **体态识别**：2秒内完成关键点检测，计算三个角度。
5. **问题标注**：在照片上标注骨骼点和问题部位（文字+箭头）。
6. **选择教练**：用户选择教练风格（鼓励/严厉/幽默）和性别。
7. **输入目标**：用户输入训练目标（如“改善圆肩”）和身体状态（下拉选择）。
8. **生成计划**：Coze Bot生成每日训练计划，展示在对话界面。
9. **开始训练**：用户点击动作卡片，跳转到B站视频跟练。
10. **打卡反馈**：用户回到对话界面，点击“做完了”或“太累了”。
11. **计划调整**：Bot根据反馈调整后续计划。

### 8.2 UI要点
- **拍照界面**：大尺寸摄像头预览，清晰的拍摄指引（如“请侧身站立”）。
- **标注界面**：半透明覆盖层，骨骼点用彩色圆点，问题部位用红色箭头和文字说明。
- **对话界面**：类聊天界面，Bot消息和用户消息分左右显示，动作卡片可点击。
- **教练选择**：卡片式选择，显示教练形象和风格描述。

### 11.3 角度计算完整示例
```javascript
// 从keypoints中提取坐标
function getPoint(keypoints, index) {
  return { x: keypoints[index].position.x, y: keypoints[index].position.y };
}

// 计算头前伸角
const ear = getPoint(keypoints, 3); // 左耳
const shoulder = getPoint(keypoints, 5); // 左肩
const neck = { x: (keypoints[0].position.x + shoulder.x) / 2, y: (keypoints[0].position.y + shoulder.y) / 2 };
const forwardHeadAngle = calculateAngle(ear, shoulder, neck);

// 计算圆肩角
const elbow = getPoint(keypoints, 7); // 左肘
const hip = getPoint(keypoints, 11); // 左髋
const roundedShoulderAngle = calculateAngle(shoulder, elbow, hip);

// 计算骨盆前倾角
const knee = getPoint(keypoints, 13); // 左膝
const ankle = getPoint(keypoints, 15); // 左踝
const anteriorTiltAngle = calculateAngle(hip, knee, ankle);
```
