# PostureFit Coze Bot 参数补充文档

> 本文档补充 Coze Bot Start 节点参数表，包含体态分析系统完整参数定义。
> 基于 MediaPipe BlazePose 33 点实现。

---

## 1. Start 节点基础参数

| 变量名 | 类型 | 必填 | 默认值 | 说明 |
|--------|------|------|--------|------|
| `score` | Number | 是 | - | 综合体态评分（0-100 分） |
| `primaryIssue` | String | 否 | - | 主要体态问题（问题类型见第2节） |
| `issues` | Array | 否 | - | 检测到的所有问题（详见第3节） |
| `coachStyle` | String | 是 | `encouraging` | 教练风格 |
| `coachGender` | String | 否 | `female` | 教练性别 |
| `coachName` | String | 是 | `爱可` | 统一教练名称 |
| `userGoal` | String | 否 | `""` | 用户目标 |
| `bodyState` | String | 是 | `normal` | 身体状态 |
| `feedback` | String | 否 | `""` | 用户打卡反馈 |
| `mode` | String | 是 | `plan` | 模式：`plan`=生成计划，`feedback`=处理反馈 |

---

## 2. 体态问题类型

系统支持检测 **9 种体态问题**（不含骨盆前倾），基于 MediaPipe BlazePose 33 点角度计算，分为正面视角（6种）和侧面视角（3种）。

### 2.1 正面视角问题

| 问题类型 | 中文名称 | 计算方式 |
|----------|----------|----------|
| `forwardHead` | 头前伸 | 耳-肩与水平线夹角（CVA近似） |
| `shoulderImbalance` | 高低肩 | 双肩连线与水平线夹角（atan2斜率） |
| `pelvicTilt` | 骨盆侧倾 | 双髋连线与水平线夹角 |
| `kneeValgus` | 膝内扣 | FPPA + 膝点偏移距离 |
| `headOffset` | 头部偏移 | 头部中线与躯干中线偏移 |
| `centerOfGravityShift` | 重心偏移 | 身体中轴与足部支撑中心偏移 |

### 2.2 侧面视角问题

| 问题类型 | 中文名称 | 计算方式 |
|----------|----------|----------|
| `roundedShoulder` | 圆肩 | 肩前移 + 躯干前倾综合评分 |
| `hunchback` | 驼背倾向 | FHP + 肩前移 + 躯干前倾加权 |
| `kneeHyperextension` | 膝超伸 | 髋-膝-踝角度 |

> **说明：** 所有计算基于 MediaPipe BlazePose 33 点技术规格，详见 `docs/MediaPipe_BlazePose_体态识别替换技术文档.md`。

## 3. 严重程度阈值

技术文档建议的角度型阈值：

| 问题类型 | 正常 | 轻度 | 中度 | 严重 |
|----------|------|------|------|------|
| `forwardHead` | >= 50° | 45° - 50° | 40° - 45° | < 40° |
| `roundedShoulder` | < 20° | 20° - 25° | 25° - 30° | > 30° |
| `kneeHyperextension` | 170° - 185° | 165° - 170° | 160° - 165° | < 160° |
| `shoulderImbalance` | < 2° | 2° - 5° | 5° - 12° | > 12° |
| `pelvicTilt` | < 2° | 2° - 5° | 5° - 12° | > 12° |
| `kneeValgus` | < 5° | 5° - 10° | 10° - 15° | > 15° |
| `headOffset` | < 3° | 3° - 5° | 5° - 8° | > 8° |
| `centerOfGravityShift` | < 3° | 3° - 5° | 5° - 8° | > 8° |
| `hunchback` | < 3° | 3° - 5° | 5° - 8° | > 8° |

> **阈值说明：** 头前伸角度越大越正常（CVA角）；膝超伸为范围型；其余为角度越大问题越严重。

### 3.1 扣分权重

| 严重程度 | 扣分 |
|----------|------|
| `normal` | 0 分 |
| `mild` | 15 分 |
| `moderate` | 25 分 |
| `severe` | 40 分 |

**综合评分计算：** `100 - 所有问题扣分总和`，最低为 0 分。

## 7. 模式配置

### 7.1 运行模式

| 模式值 | 中文名称 | 说明 |
|--------|----------|------|
| `plan` | 生成计划 | 根据分析结果生成训练计划 |
| `feedback` | 处理反馈 | 处理用户打卡反馈 |

### 7.2 拍摄模式

| 模式值 | 中文名称 | 可分析的问题数量 |
|--------|----------|------------------|
| `fullBody` | 全身 | 全部 10 种问题 |
| `halfBody` | 半身为 | 6 种问题（无膝/脚踝相关） |
| `closeUp` | 局部特写 | 4 种问题（仅头颈肩） |
| `sitting` | 坐姿 | 6 种问题（无脚踝相关） |

### 7.3 视角选择

| 视角值 | 中文名称 | 可分析的问题 |
|--------|----------|--------------|
| `front` | 正面 | 6 种正面问题 |
| `side` | 侧面 | 4 种侧面问题 |
| `dual` | 双视角 | 全部 10 种问题 |
