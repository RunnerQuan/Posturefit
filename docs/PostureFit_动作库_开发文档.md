## 使用说明

### 1. 数据结构
- 每个工作表对应一个体态问题分类
- 每个动作包含：id、name、description、videoUrl、safetyTips、targetMuscles、difficulty、duration、repetitions、sets、source、tags

### 2. 字段说明
| 字段 | 类型 | 说明 |
|------|------|------|
| id | string | 动作唯一标识符，如 fh-1、rs-1 |
| name | string | 动作名称 |
| description | string | 动作描述，详细说明如何做 |
| videoUrl | string | B站视频链接 |
| safetyTips | string | 安全提示，注意事项 |
| targetMuscles | string | 目标肌肉，用逗号分隔 |
| difficulty | string | 难度等级：beginner/intermediate/advanced |
| duration | number | 时长（秒） |
| repetitions | number | 重复次数 |
| sets | number | 组数 |
| source | string | 专业来源，如康复师、课程等 |
| tags | string | 场景标签，用逗号分隔 |

### 3. 动作ID命名规则
| 前缀 | 体态问题 |
|------|----------|
| fh | 头前伸矫正 |
| rs | 圆肩矫正 |
| ap | 骨盆前倾矫正 |
| si | 高低肩矫正 |
| pt | 骨盆侧倾矫正 |
| kv | 膝内扣矫正 |
| ho | 头部偏移矫正 |
| cg | 重心偏移矫正 |
| hb | 驼背倾向矫正 |
| kh | 膝超伸矫正 |

### 4. 使用示例
```javascript
// 根据体态问题获取动作列表
const getActionsByProblem = (problem) => {
  const actionLibrary = { /* JSON数据 */ };
  return actionLibrary[problem] || [];
};

// 获取特定难度的动作
const getActionsByDifficulty = (problem, difficulty) => {
  const actions = getActionsByProblem(problem);
  return actions.filter(action => action.difficulty === difficulty);
};

// 生成训练计划
const generateTrainingPlan = (problem, count = 3) => {
  const actions = getActionsByProblem(problem);
  return actions.slice(0, count);
};
```
